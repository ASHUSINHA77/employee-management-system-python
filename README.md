# 👨‍💼 Employee Management System

A **Python-based Employee Management System** that allows users to efficiently manage employee records through a simple **menu-driven interface**. The application demonstrates core Python concepts including **Object-Oriented Programming (OOP), File Handling, Functions, and Exception Handling**.

---

## 🚀 Features

- ➕ Add Employee
- ✏️ Update Employee Details
- ❌ Delete Employee
- 🔍 Search Employee
- 📋 View All Employees
- 💾 Store Employee Records Using File Handling
- ⚠️ Exception Handling for Invalid Inputs
- ✅ Input Validation for Reliable Data Entry
- 🖥️ Easy-to-Use Menu-Driven Interface

---

## 🛠️ Tech Stack

- 🐍 Python 3
- 📂 File Handling
- 🏗️ Object-Oriented Programming (OOP)
- ⚙️ Functions
- 🚨 Exception Handling
